# Plant Points Tracker 🌿

A simple, interactive desktop application built with Python and Kivy to help track plant consumption.

## Features
* **Search & Log:** Easily search for fruits, vegetables, and other plants to add to your daily log.
* **Point Counter:** Automatically calculates and displays your current "Plant Points" based on what you have eaten.
* **Weekly History:** Keeps a running list of the specific plants you have consumed this week and how many times you've eaten them.
* **Visual Heatmap:** Displays a visual grid (heatmap) to track your plant-eating streaks and habits over time.

## Data Storage
Your personal eating habits are saved locally on your machine in an `eaten_plants_data.json` file.
